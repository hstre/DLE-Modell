# Dual-Layer Economy (DLE) Reference Model

Reference implementation of the Dual-Layer Economy (DLE), a stock-flow consistent macroeconomic architecture designed to study structural resilience under severe shocks.

This repository contains the exact simulation code used in the DLE working paper.

## Key features

- Stock-flow consistent macroeconomic simulation
- Endogenous resilience capital (Z)
- Deterministic recovery regime simulation
- Invariant-enforced accounting structure
- Full reproducibility of published results

## Requirements

Python 3.10+

Install dependencies:

pip install numpy pandas matplotlib

## Run simulation

python simulate_recovery_regime.py

## Purpose

This model is an architectural proof-of-concept for resilience-oriented macroeconomic system design.

It is not intended as a predictive model of any specific economy.
